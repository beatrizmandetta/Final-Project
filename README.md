# Data Science and Python Final Project

Final project analysis for identifying major-city air routes that may be underserved relative to population, tourism, and country-level economic demand.

## Main Files

- `Final_Underserved_Intercity_Routes.ipynb`: polished final Python notebook with data loading, cleaning, GDP-enhanced EDA, models, and conclusion.
- `final_project_sql_queries.sql`: separate SQL deliverable with 12 commented SQLite queries.
- `final_project_routes.sqlite`: SQLite export generated by the notebook for running the SQL queries.
- `Final_Project_Presentation_Summary.pdf`: concise presentation summary.
- `data/country_gdp_current_usd_clean.csv`: cleaned World Bank GDP data used for analysis.
- `data/aviationstack_flights_cached.csv`: cached AviationStack sample so the notebook can rerun without another API request.
- `data/world_bank_tourism_cached.csv`: cached World Bank tourism indicators.

## Data Sources

- AviationStack API: observed flight records.
- OurAirports `airports.csv`: airport geography and IATA codes.
- UN WUP top-100 city file: city population and coordinates.
- World Bank tourism indicators: country-level destination tourism demand.
- World Bank GDP indicator `NY.GDP.MKTP.CD`: country-level economic capacity.
